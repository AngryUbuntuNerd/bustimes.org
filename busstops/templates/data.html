Data sources

This website uses information from several third-party open data sources, licensed under version 3 of the Open Government License.

National Public Transport Gazetteer

The NPTG defines every region, administrative area, district and locality in Great Britain.

National Public Transport Access Node database

The NaPTAN database contains every bus stop, railway station, ferry terminal, etc in Great Britain. It includes their locations in terms of both geographic coordinates and NPTG localities.

National Operator Codes

The NOC database contains every public transport operator in Great Britain.

Traveline National Dataset

The TNDS contains timetables for every bus, light rail, tram and ferry service in Great Britain. Services are linked to NaPTAN stops and NOC operators. The dataset is updated roughly every week, and we automatically update this website within a few hours each time.

Transport for London

We use TFL open data to augment the NaPTAN data, and to provide live departure information for bus stops in London.

Transport API

We use Transport API to provide departure information for some bus stops.
